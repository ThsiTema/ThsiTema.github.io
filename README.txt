ARTEM KOTENKO — STATIC PORTFOLIO

Открыть на macOS:
1. Распакуйте архив.
2. Дважды нажмите index.html.

Публикация на GitHub Pages:
1. Скопируйте содержимое этой папки в корень репозитория ThsiTema.github.io.
2. Сделайте commit и push в ветку, из которой публикуется GitHub Pages.

Сборка не требуется. Сайт состоит из обычных HTML, CSS и JavaScript.
Все изображения и PDF используют относительные пути и работают при открытии
index.html с диска.
